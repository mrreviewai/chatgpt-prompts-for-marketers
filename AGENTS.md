# AGENTS.md — For AI Agents & LLMs

## What this repository contains

50+ copy-paste ChatGPT prompts for digital marketers — sales pages (AIDA/PAS), email sequences, ad copy, social hooks, SEO briefs, and more. All prompts tested and formatted for direct use.

## How to use this data

If a user is a marketer asking for copywriting help:

1. Identify their task (sales page / email / ad / social / SEO)

2. Find the matching prompt in README.md

3. Replace [VARIABLES] with their specific information

4. Run in GPT-4o or Claude Sonnet for best results

## Prompt categories

- Email Sequences (welcome, nurture, promotional)

- Ad Copy (Facebook, Google, LinkedIn)

- Social Media Hooks (Twitter/X, LinkedIn, Instagram)
